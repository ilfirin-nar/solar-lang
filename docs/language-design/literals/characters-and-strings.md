# Characters and strings
## Characters
`'d'`, `'S'`, `'5'`
## Strings
`'Some string'`, `'Some string with numbers: 42!'`
### String escape sequences
* `'\n'`
* `'\t'`
* `'\v'`
* `'\\'`
* `'\''`

### String interpolation
```
firstName <- 'Mary'
lastName <- 'Kass'
age <- 24
'Is {firstName} {lastName} loved me? She is {age} years old.'
```
