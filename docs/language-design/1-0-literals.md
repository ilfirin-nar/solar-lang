# Literals
## Numbers
* `42` — integer number
* `3.14` — real number
* `0x4fd2` — HEX form of number

## Characters and strings
### Characters
`'d'`, `'S'`, `'5'`
### Strings
`'Some string'`, `'Some string with numbers: 42!'`
#### String escape sequences
* `'\n'`
* `'\t'`
* `'\v'`
* `'\\'`
* `'\''`

#### String interpolation
```
firstName <- 'Mary'
lastName <- 'Kass'
age <- 24
'Is {firstName} {lastName} loved me? She is {age} years old.'
```
